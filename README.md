## Project Description

CourseHub là website giới thiệu các khóa học lập trình dành cho sinh viên.

Dự án được xây dựng nhằm thực hành các kiến thức về:

- HTML
- CSS
- JavaScript
- Bootstrap 5
- Git & GitHub Workflow

Website cho phép người dùng:

- Xem danh sách khóa học
- Tìm kiếm khóa học
- Lọc khóa học theo cấp độ
- Xem chi tiết khóa học bằng Modal
- Đăng ký khóa học
- Lưu thông tin đăng ký bằng LocalStorage
- Xem danh sách học viên đã đăng ký

## Technologies

## Features

### Navbar & Hero Section
- Thanh điều hướng
- Banner giới thiệu CourseHub

### Quản lý khóa học
- Hiển thị danh sách khóa học
- Tìm kiếm theo tên khóa học
- Lọc theo cấp độ
- Modal xem chi tiết

### Đăng ký khóa học
- Nhập họ tên
- Nhập email
- Kiểm tra dữ liệu đầu vào

### Local Storage
- Lưu danh sách đăng ký
- Hiển thị lại khi tải trang
- Xóa đăng ký
